Help

Enter your code
If you have already been given a code to access a file share location, enter it into the text box and press Go. Please contact the associate you are working with if you need to share files and do not have a code.
You may have to provide a user name and password to access the page. If you cannot remember this information, please contact us.

Click Browse... and find your file to upload
Under the Share a New File heading, click the Browse... button. A new window will appear, prompting you to find a file on your computer to upload.
After you have found the file, press the Open button at the lower-right of the window. It will disappear, and your file name will be displayed in the text box next to the Browse... button.

Click Upload
Make sure the file you wish to upload is displayed in the text box, and click the Upload button. You will be able to monitor the progress of your upload as it happens.
After the upload is finished, your file will be included in the List of Shared Files at the top of the page. It will be immediately available for anyone to download.

Share the uploaded file
There are a few ways to share your uploaded files:

  • Highlight and copy the Shared File Location's code under the About this File Sharing Location heading, and give it to the intended recepient.
  • Highlight and copy/paste the link to the File Sharing Location (also under the About this File Sharing Location heading). This would enable the recipient to access the File Sharing Location without entering a code.
  • Highlight and copy/paste the direct link to the file you wish to share, located in the List of Shared Files.

Deleting Files
To delete a file, click the check box next to the unwanted file, and press the delete button. You can delete all files from the share location by placing a check in the box at the top of the list.
